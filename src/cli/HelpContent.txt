Here are the options available in the application

Use java -jar gui.jar
[-h or --help] Provide description of the available options for the application.

[-d <Directory path>] Display all images of type PNG, JPG and JPEG in the provided directory.

[-f <Image path>] Display the EXIF data of the provided image.

[-s <ASCII text>] Specify a text to hide in an image. (require -f option)

[-e] Extract a message from an image. (require -f option)

You also can run the graphical version of the application by using java -jar gui.jar
